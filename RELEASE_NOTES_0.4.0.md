# WAFD ONE 0.4.0

- توسعة جميع النماذج الأساسية مع الحفاظ على الحقول القديمة.
- إضافة فنادق المشروع كجدول فرعي.
- إضافة الإنتاج، الجودة، التحميل، الشكاوى، وتكاليف المشروع.
- ربط دورة التشغيل كاملة والتحقق من الكميات والتواريخ والسعة.
- تحديث حالة الخطة والمركبة والسائق تلقائيًا عند الاعتماد والتسليم.
- إضافة مؤشرات المشروع: إجمالي الوجبات، المسلّم، المتبقي، الإنجاز، التكلفة والربح.
- تحديث غرفة العمليات والاختصارات.
