# WAFD ONE Enterprise 10.0.0 RC1

Release candidate for user acceptance testing.

- Modern enterprise UI layer for Desk forms, lists and mobile views.
- New Launch Center with live metadata and master-data readiness checks.
- Unified UAT path from Contract through Payment.
- Dynamic synchronization of every shipped DocType, preventing omissions in future migrations.
- Daily planning, multi-warehouse allocation, production, quality, packaging, loading, delivery, invoicing and payment remain integrated.
- Safe migration that reloads metadata, Launch Center and Workspace without deleting operational records.
- External-provider features such as live GPS and WhatsApp are intentionally not claimed as active until provider credentials are configured.
