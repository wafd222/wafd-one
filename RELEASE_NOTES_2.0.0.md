# WAFD ONE v2.0.0

## النسخة التشغيلية الأولى لواجهة النظام

- إنشاء Workspace مكتمل بدل الصفحة الفارغة.
- إضافة 22 اختصارًا تشغيليًا مرتبة حسب دورة العمل.
- إضافة بطاقات أقسام للمشاريع والإنتاج والتوزيع والأسطول والمخزون والمالية.
- مزامنة Workspace تلقائيًا بعد كل تثبيت أو تحديث لمنع بقاء محتوى قديم أو فارغ.
- الإبقاء على الصلاحيات التشغيلية الثمانية وتطبيقها على مساحة العمل.
- تحديث ترتيب القوائم ليتوافق مع Frappe v16 باستخدام تاريخ الإنشاء.
- دعم العرض العربي/الإنجليزي في النماذج الحالية.

## دورة التشغيل

المشروع ← خطة الوجبات ← الإنتاج ← الجودة ← التحميل ← رحلة التوصيل ← إثبات التسليم ← التكاليف والإيرادات والفواتير.
