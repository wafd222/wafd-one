# WAFD ONE 3.2.0 — Production Operations

- Create a production batch directly from an approved meal plan.
- Copy project, recipe, date and planned quantity into the production batch.
- Generate a draft material issue from recipe ingredients and recipe yield.
- Link the generated stock movement back to the production batch.
- Create and reuse a quality inspection from the production batch.
- Synchronize meal-plan status with production progress.
- Stop a production batch automatically when quality inspection is rejected.
- Calculate actual production yield percentage.
