# WAFD ONE 5.1.0

- إضافة شاشة إعدادات طباعة مركزية داخل النظام.
- التحكم بالشعار، العلامة المائية، الألوان، الخطوط، الهوامش، حجم الورق، التوقيع والختم.
- معاينة تعهد سابق مباشرة من شاشة الإعدادات.
- ربط قالب تعهد الفندق بالإعدادات ديناميكيًا دون تعديل الكود.
- إضافة اختصار إعدادات الطباعة إلى مساحة عمل WAFD ONE.
