# WAFD ONE 5.3.0 — Contract-Driven Operations

## تنفيذ دورة التشغيل من العقد
- جعل حقل المشروع في العقد اختياريًا حتى يمكن إنشاء المشروع من العقد دون تعارض.
- إضافة نوع المشروع والخدمات التشغيلية وإعدادات المستودع والمركبة والسائق إلى العقد.
- نسخ الفنادق والخدمات والإعدادات إلى المشروع عند إنشائه.
- إضافة أمر واحد لتفعيل العقد وإنشاء المشروع وخطط الوجبات ودفعات الإنتاج.
- التنفيذ آمن عند التكرار ولا ينشئ سجلات تشغيل مكررة.

## إصلاحات حرجة
- تعيين نوع المشروع الإلزامي عند إنشاء المشروع من العقد.
- التحقق من بيانات العقد قبل التفعيل.
- منع العقد الملغي من إنشاء دورة تشغيل.
- عدم إعادة كتابة خدمات مشروع بدأ تشغيله فعليًا.
- Patch آمن لتصحيح نوع المشروع والروابط أحادية الاتجاه في البيانات القديمة.
