# WAFD ONE 6.0.0 — Governance, Approvals and Immutable Audit

- Optional amount-threshold approval controls for contracts, purchase orders, invoices, payments, costs and revenues.
- Maker-checker segregation with self-approval prevention.
- Immutable approval requests with document snapshots and decision records.
- Immutable audit events for critical operational and financial documents.
- Governance health-check API and duplicate-pending-approval detection.
- Safe defaults: approval gates are disabled until explicitly enabled by management; audit logging is enabled.
- New roles: WAFD Approver and WAFD Auditor.
