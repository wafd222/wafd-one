# WAFD ONE v6.1.0

Finance integrity and controlled billing release.

- Keeps invoice quantities tied to accepted delivery proofs and blocks over-invoicing.
- Closes direct-confirmation bypasses for governed invoices and payments.
- Requires governed documents to be saved as drafts before approval is requested.
- Adds `finance_integrity_check` for invoice/payment/delivery reconciliation.
- Detects over-invoiced meal plans, duplicate invoice rows, payment overruns, and balance mismatches.
- Adds a safe repair mode that recalculates invoice balances, statuses, and project financial summaries without creating or deleting transactions.
- Updates package and application version metadata to 6.1.0.
