# WAFD ONE 6.7.0

## Document Studio Professional Editor
- True drag and drop from the elements library.
- A4 canvas with zoom, rulers, grid and snap.
- Keyboard shortcuts for save, copy, paste, duplicate and delete.
- Element locking and layer controls.
- Rich element properties: position, size, font, size, color, background, opacity and rotation.
- Searchable grouped dynamic fields.
- Improved responsive three-panel interface.
- Canvas schema upgraded to version 2 while remaining backward-compatible.
