# WAFD ONE v7.6.0

## Food safety, quality and traceability

- Added a focused food-safety dashboard service with release, inspection and CCP KPIs.
- Added end-to-end batch traceability lookup using the production traceability code.
- Added hourly alerts for unresolved CCP deviations and pending food-safety releases.
- Added early warning when an unreleased batch is within four hours of service.
- Corrected release logic so a documented, verified corrective action can close a CCP deviation.
- Added migration support to populate traceability codes on legacy production batches.
