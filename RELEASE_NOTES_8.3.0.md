# WAFD ONE v8.3.0

## Daily Meal Planning and Production Automation

- Added **WAFD Daily Meal Plan**: one operational record per project, hotel and service date.
- Added a child meal table for breakfast, lunch, dinner, suhoor, iftar and coffee break.
- Consolidates the proven v8.2 individual meal-plan records without deleting or invalidating them.
- Automatically calculates daily quantities, revenue, estimated cost, profit and margin.
- Loads recipe cost automatically and reports meals that still require recipes.
- Adds duplicate protection for project/hotel/date and meal type.
- Preserves daily plans that have already entered production.
- Creates or reuses one production batch for every meal row.
- Links every daily row to its operational meal plan and production batch for traceability.
- Carries the project's kitchen and source warehouse into daily planning and production.
- Adds project buttons for generating daily plans and viewing a planning summary.

## Upgrade safety

This release is additive. Existing WAFD Meal Plan and WAFD Production Batch records remain supported and are not deleted during migration.
